## Laravel 11 + Sanctum + Vue 3.5 + Vuetify 3

### 1. Clone project
### 2. Install PHP libraries 
```
composer install
```
### 3. Create .env file
```
cp .env.example .env
```
### 4. Generate key
```
php artisan key:generate
```
### 5. Modify .env file with necessary data
### 6. Install node modules
```
npm i
```
### 7. Run dev or production
```
npm run dev
```
```
npm run build
```
